이 원장, 당신이 직접 검증하세요

우리를 믿지 마세요. 아래 버튼을 누르면 당신의 브라우저가 공개 원장을 받아 제네시스부터 SHA-256 해시체인을 다시 계산하고, 한 줄이라도 위조·삭제·재배열됐는지 스스로 확인합니다. 서버도, 설치도, 우리에 대한 신뢰도 필요 없습니다.

원장(changes.jsonl)과 체인 상태(integrity.json)를 jsDelivr로 받아 재계산합니다.
준비됨.

검증 알고리즘(엔진과 동일): content_hash = SHA256(canonicalJson(본문 6필드)) · chain_hash = SHA256(prev + content_hash) · genesis = SHA256("chronicle:<id>:genesis"). 외부 앵커(RFC 3161)는 openssl ts -verify로 별도 검증됩니다 — 리포 소유자도 위조할 수 없는 층.
